# ClassHub - Classroom Reservation System

A centralized classroom reservation system for managing reservation requests and displaying the latest schedules. This project is developed as part of an undergraduate thesis.

## About

ClassHub is a web-based application designed to streamline the classroom reservation process in educational institutions. The system centralizes all reservation requests, making it easier for administrators to manage bookings and for users to view real-time classroom availability and schedules.

## Features

- **Centralized Reservation Management**: Submit and manage classroom reservation requests through a unified platform
- **Real-time Schedule Display**: View the latest classroom schedules and availability
- **Request Processing**: Streamlined workflow for handling reservation requests

## Technology Stack

- **Backend**: Laravel 12 (PHP 8.2+)
- **Frontend**: Laravel Blade & Livewire templates with Tailwind CSS
- **Database**: PostgreSQL (configurable)
- **Build Tool**: Vite

## Requirements

- PHP >= 8.2
- Composer
- Node.js and npm
- Database server (MySQL, PostgreSQL, or SQLite)
